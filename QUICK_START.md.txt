# 🚀 iCoder Plus v2.0 - Quick Start Guide

**AI-first IDE in a bottom sheet** - Get up and running in 5 minutes!

## ⚡ Instant Setup (Recommended)

### Option 1: Automated Script
```bash
# Make setup script executable and run
chmod +x setup.sh
./setup.sh
```

### Option 2: Manual Setup
```bash
# 1. Install dependencies
npm install

# 2. Copy environment template
cp .env.example .env

# 3. Add your OpenAI API key to .env
# VITE_OPENAI_API_KEY=your_key_here

# 4. Start development server
npm run dev
```

## 📋 Prerequisites

- **Node.js** 18+ ([Download](https://nodejs.org))
- **npm** 9+ (comes with Node.js)
- **OpenAI API Key** ([Get one here](https://platform.openai.com/api-keys))

## 🔑 API Key Setup

1. Open `.env` file
2. Replace `your_openai_api_key_here` with your actual API key:
```env
VITE_OPENAI_API_KEY=sk-your-actual-key-here
```

## 🔧 Troubleshooting

### Common Issues:

**"API key not configured"**
- Check `.env` file exists and contains valid OpenAI key
- Restart dev server after adding API key

**Dependencies error**
- Delete `node_modules/` and `package-lock.json`
- Run `npm install` again

**Port 5173 in use**
- Change port in `vite.config.js`: `server: { port: 3000 }`

**Build fails**
- Check all imports in source files are correct
- Ensure all required files are present
