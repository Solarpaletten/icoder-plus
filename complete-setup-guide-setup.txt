## ğŸ”§ Troubleshooting

### Common Issues:

**"API key not configured"**
- Check `.env` file exists and contains valid OpenAI key
- Restart dev server after adding API key

**Dependencies error**
- Delete `node_modules/` and `package-lock.json`
- Run `npm install` again

**Port 5173 in use**
- Change port in `vite.config.js`: `server: { port: 3000 }`

**Build fails**
- Check all imports in source files are correct
- Ensure all required files are present
